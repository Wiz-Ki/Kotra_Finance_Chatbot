11.1 이체비목 발생 상황사례 및 입력방법
========================

  

| 이체비목 발생 상황 | 이체비목 발생 상황별 입력방법 | 입금 시 사례 | 지출 시 사례 | 이체비목 발생 상황 사례 |
=====================================================================

  

|:---|:---|:---|:---|:---|  
| 예금장부에 입금하고 예금장부에서 지출하는 경우 | 1. 예금 입금 시 : 예금장부에, 비목은 '이체'로, 입출구분은 '입금'으로 입력  
2. 예금  
지출 시 : 예금장부에, 비목은 '이체'로, 입출구분은 '출금'으로 입력 | 외부기관 가수금 또는 BI 분담금을 무역관 계좌에 입금하고, 무역관  
계좌에서 지출하는 경우 | 단순 입금 및 지출 오류로 인해 무역관 계좌에 입금하고, 무역관 계좌에서 지출하는 경우 | 주택임차료(또는  
자녀교육비, 의료보험료, 개인차량보험료 등) 개인부담금을 무역관 계좌에 입금하고, 무역관 계좌에서 지출하는 경우 |  
| 현금장부에 입금하고 현금장부에서 지출하는 경우 | 1. 현금 입금 시 : 현금장부에, 비목은 '이체'로, 입출구분은 '입금'으로 입력  
2. 현금  
지출 시 : 현금장부에, 비목은 '이체'로, 입출구분은 '출금'으로 입력 | 외부기관 가수금 또는 BI 분담금을 현금으로 입금하고, 현금으로  
지출하는 경우 | 단순 입금 및 지출 오류로 인해 현금으로 입금하고, 현금으로 지출하는 경우 | 주택임차료(또는 자녀교육비, 의료보험료,  
개인차량보험료 등) 개인부담금을 현금으로 입금하고, 현금으로 지출하는 경우 |  
| 예금장부에 입금하고 현금장부에서 지출하는 경우 | 1. 예금 입금 시 : 예금장부에, 비목은 '이체'로, 입출구분은 '입금'으로 입력  
2. 현금  
지출 시  
 1) 예금장부에, 비목은 '이체'로, 입출구분은 '출금'으로 입력  
 2) 장부대체로, 현금장부-출금, 예금장부-입금으로 입력  
(장부 대체시에는 비목 입력 불요) | 외부기관 가수금 또는 BI 분담금을 무역관 계좌에 입금하고, 현금으로 지출하는 경우 | 단순 입금 및  
지출 오류로 인해 무역관 계좌에 입금하고, 현금으로 지출하는 경우 | 주택임차료(또는 자녀교육비, 의료보험료, 개인차량보험료 등)  
개인부담금을 무역관 계좌에 입금하고, 현금으로 지출하는 경우 |  
| 현금장부에 입금하고 예금장부에서 지출하는 경우 | 1. 현금 입금 시 : 현금장부에, 비목은 '이체'로, 입출구분은 '입금'으로 입력  
2. 예금  
지출 시   
 1) 현금장부에, 비목은 '이체'로, 입출구분은 '출금'으로 입력  
 2) 장부대체로, 예금장부-출금, 현금장부-입금으로 입력  
(장부 대체시에는 비목 입력 불요) | 외부기관 가수금 또는 BI 분담금을 현금으로 입금하고, 무역관 계좌에서 지출하는 경우 | 단순 입금  
및 지출 오류로 인해 현금으로 입금하고, 현금으로 지출하는 경우 | 주택임차료(또는 자녀교육비, 의료보험료, 개인차량보험료 등)  
개인부담금을 현금으로 입금하고, 무역관 계좌에서 지출하는 경우 |  
| 카드장부에서 지출하고 카드장부에서 지출이 취소되는 경우 | 1. 카드 지출 시 : 카드장부에, 비목은 '이체'로, 입출구분은 '출금'으로  
입력  
2. 카드 취소 시 : 카드장부에, 비목은 '이체'로, 입출구분은 '입금'으로 입력 | | 실수로 개인사용분을 공용카드로 결제하여,  
카드결제를 취소한 경우 | 공용카드로 물품을 구매했는데, 재고부족 또는 변심 등으로 주문이 취소됨에 따라 카드결제가 취소된 경우 |  
| 예금장부에 입금하고 신용카드장부에서 지출하는 경우 | 1. 예금 입금 시 : 예금장부에, 비목은 '이체'로, 입출구분은 '입금'으로 입력  
2.  
신용카드 지출 시 : 신용카드장부에, 비목은 '이체'로, 입출구분은 '출금'으로 입력  
3. 신용카드 대금 결제 시  
 1) 예금장부에,  
비목은 '이체'로, 입출구분은 '출금'으로 입력  
 2) 신용카드장부에, 비목은 '이체'로, 입출구분은 '입금'으로 입력  
 3) 신용카드 대금  
대체 입력 시(장부대체로, 예금장부-출금, 카드장부-입금으로 입력), 위에서 입력한 이체비목 금액을 차감한 금액으로 입력 (장부  
대체시에는 비목 입력 불요) | 실수로 개인사용분을 공용카드로 결제하여, 해당 금액을 무역관 계좌로 환입한 경우 | | 공용카드로 물품을  
구매했는데, 재고부족 또는 변심 등으로 주문이 취소됨에 따라 구매대금이 무역관 계좌로 환입된 경우 |  
| 현금장부에 입금하고 신용카드장부에서 지출하는 경우 | 1. 현금 입금 시 : 현금장부에, 비목은 '이체'로, 입출구분은 '입금'으로 입력  
2.  
신용카드 지출 시 : 신용카드장부에, 비목은 '이체'로, 입출구분은 '출금'으로 입력  
3. 신용카드 대금 결제 시  
 1) 현금장부에,  
비목은 '이체'로, 입출구분은 '출금'으로 입력  
 2) 신용카드장부에, 비목은 '이체'로, 입출구분은 '입금'으로 입력  
 3) 장부대체로,  
신용카드장부-출금, 현금장부-입금으로 입력 (장부 대체시에는 비목 입력 불요) | 실수로 개인사용분을 공용카드로 결제하여, 해당  
금액을 현금으로 환입한 경우 | | 공용카드로 물품을 구매했는데, 재고부족 또는 변심 등으로 주문이 취소됨에 따라 구매대금이 현금으로  
환입된 경우 |  
| 카드장부에서 지출하였으나 카드장부에서 지출이 취소되지 않고 예금장부에 입금된 경우이면서, 지출과 입금이 동일분기에 발생되지 않아  
신용카드 대금을 전분기에 이미 결제한 경우 (카드 지출 및 대금 결제는 전분기에, 예금 입금은 본분기에 발생) | 1. 전분기 신용카드 지출  
시 : 신용카드장부에, 비목은 '이체'로, 입출구분은 '출금'으로 입력  
2. 전분기 신용카드 대금 결제 시 : 장부대체로, 예금장부-출금,  
카드장부-입금으로 입력 (이체비목 금액을 차감하지 않은 금액으로 입력한 경우)  
3. 본분기 예금 입금 시  
 1) 장부대체로,  
신용카드장부-출금, 예금장부-입금으로 입력 (장부 대체시에는 비목 입력 불요)  
 2) 신용카드장부에, 비목은 '이체'로, 입출구분은  
'입금'으로 입력 | | 전분기에 실수로 개인사용분을 공용카드로 결제하여, 해당 금액을 무역관 계좌로 본분기에 환입한 경우 | 전분기에  
공용카드로 물품을 구매했는데, 재고부족 또는 변심 등으로 주문이 취소됨에 따라 구매대금이 본분기에 무역관 계좌로 환입된 경우 |  
| 지출과 입금이 동일분기에 발생되지 않은 경우이면서, 전분기에 이체비목이 아닌 비용계정으로 정산(지출)했던 경우 | 1. 전분기 지출 시 :  
집행내역등록 화면에서 해당 장부에 비용계정으로 출금 입력  
2. 본분기 입금 시 : 환입등록 화면에서 해당 장부에 지출 당시 입력한  
비용계정으로 환입 입력 | | | |
